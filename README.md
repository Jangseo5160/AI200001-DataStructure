
Python 표준 라이브러리만 사용하는 턴제 던전 게임 초안

## 조작법

| 키 | 기능 |
|---|---|
| w | 위로 이동 |
| a | 왼쪽 이동 |
| s | 아래 이동 |
| d | 오른쪽 이동 |
| space | 인접한 적 공격 |
| p | 포션 사용 |
| u | Undo |
| l | 리더보드 보기 |
| q | 종료 |

## 기호

| 기호 | 의미 |
|---|---|
| @ | 플레이어 |
| # | 벽 |
| . | 바닥 |
| P | 포션 |
| S | 검 |
| X | 적 |
| G | 출구 |
